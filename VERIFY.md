# iPhone 真机验收清单

部署:把整个目录放到一个 HTTPS 静态托管(GitHub Pages / Vercel / Cloudflare Pages 等)。
PWA 的 service worker 与"添加到主屏幕"需要 HTTPS(localhost 例外)。

## 安装
- [ ] Safari 打开站点,底部出现"添加到主屏幕"提示条
- [ ] 分享 → 添加到主屏幕,主屏幕出现 iPod 图标
- [ ] 从图标打开:全屏,无 Safari 地址栏/工具栏
- [ ] 安装后提示条不再出现

## 导入与持久化
- [ ] 点右上 ＋ 选几首本地歌(mp3/m4a),导入成功
- [ ] 主菜单→歌曲,能看到列表与歌名/歌手(文件名"艺人 - 标题"被正确拆分)
- [ ] 完全关掉再从图标打开,歌还在

## 轮盘交互
- [ ] 菜单画圈:顺时针下移、逆时针上移,边界停住
- [ ] 中心键进入歌曲列表;列表画圈滚动,转得快滚得快
- [ ] 中心键播放选中歌,进入正在播放
- [ ] MENU 逐级返回:正在播放→列表→主菜单
- [ ] 滚动有轻微震动

## 正在播放
- [ ] 背景是该歌专属撞色渐变,歌名/歌手清晰可读
- [ ] 同一首歌每次打开颜色一致
- [ ] ⏯(下)播放/暂停;长按中心圆也能播放/暂停
- [ ] ⏭/⏮ 切歌;歌曲开头 3 秒内按 ⏮ 回上一首,之后按 ⏮ 回到开头
- [ ] 点中心键切换 🔊/⏩ 模式,图标随之变化
- [ ] 🔊 模式画圈调音量(能听到音量变化);⏩ 模式画圈快进/快退
- [ ] 锁屏控制中心显示歌名/歌手(Media Session)

## 边界
- [ ] 无歌时显示空态"点这里导入"可用
- [ ] 离线(飞行模式)从图标仍能打开 app 外壳
